Integrate today

Enhance your applications with powerful AI agent capabilities.

// Install the package
npm install web-agent

// Initialize the agent
import { WebAgent } from 'web-agent';

const agent = new WebAgent({
  apiKey: "YOUR_API_KEY"
});

// Run a task
const result = await agent.run({
  task: "Describe what you want the agent to do"
});

console.log(result);
